# CV Website - Training Roadmap

**Goal:** Junior -> Mid-level Developer
**Time:** ~1 hour/day | **Started:** March 2026
**Stack:** React + C# ASP.NET + MongoDB

---

## Phase 1: Foundations (Week 1)

- [ ] **Day 1** — C# OOP & SOLID (classes, interfaces, models for CV data) *(in progress — continue with Education, Language, Project, Person, then Program.cs)*
- [ ] **Day 2** — ASP.NET Minimal API (first GET endpoint returning CV JSON)
- [ ] **Day 3** — JavaScript Core (functions, objects, array methods)
- [ ] **Day 4** — Intro to React (first components, JSX, props)
- [ ] **Day 5** — MongoDB Data Modelling (schema design for CV)

## Phase 2: Connecting the Pieces (Week 2)

- [ ] Connecting React to the API (fetch, async/await)
- [ ] React state management (useState, useEffect)
- [ ] Entity Framework / MongoDB driver in C#
- [ ] React routing (multi-page CV site)

## Phase 3: Making It Real (Week 3)

- [ ] CSS/styling in React (Tailwind or styled-components)
- [ ] Middleware, error handling, logging in ASP.NET
- [ ] Unit testing & TDD basics (xUnit + Jest)
- [ ] Git workflow & PR best practices

## Phase 4: Going Deeper (Week 4+)

- [ ] TypeScript introduction
- [ ] Async programming deep dive (C# Tasks / JS Promises)
- [ ] System design basics (client-server, WebSockets vs HTTP)
- [ ] DynamoDB basics (transferring MongoDB knowledge)
- [ ] Deployment (getting the site live)
